## Surrounding Travel Platform Personal Management

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/08ae8d75b5af4d98aef89f86b99ebce8/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Attractions
5. Rooms
6. Tour Groups
7. Travel Routes
8. Announcements
9. Message Board
10. Personal Center
11. Backend Management (Homepage, Personal Center, Attraction Management, Room Management, Tour Group Management, Travel Route Management, Announcement Management, Message Board Management)
12. Customer Service


**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center
4. Customer Service Chat Management
5. Basic Data Management (Room Type Management, Announcement Type Management, Attraction Type Management, Tour Group Type Management, Departure Time Management, Travel Method Management, Tourism Method Management)
6. Attraction Management (Attraction Review Management, Attraction Favorites Management, Attraction Order Management)
7. Room Management (Room Management, Room Review Management, Room Favorites Management, Room Order Management)
8. Tour Group Management (Tour Group Management, Tour Group Message Management, Tour Group Favorites Management, Tour Group Grouping Management)
9. Travel Route Management (Travel Route Management, Travel Route Message Management, Travel Route Favorites Management)
10. Announcement Management
11. Message Board Management
12. User Management
13. Carousel Information
